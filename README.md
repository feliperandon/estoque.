# estoque.

An web app focused on product, material and sales management, allowing you to control the flow of your business.

It counts with:
- Main dashboard to see costumers/sales graphs and products/materials in stock;
- Product page to add, remove or edit product, being able to put image, description, time spent, money spent, materials used, quantity;
- Material page to add, remove or edit material, you can add/edit putting name, category, quantity, supplier, metric;
- Sales page to register new sales passing the product, a client. price. Sales history aside;
- Costumers page to keep track of your clients orders, you can add and edit clients. Email and number to contact, quantity bought and its prices, personal history;


## Stacks used
- Framework: React
- TypeScript
- Styling: Tailwind
- Forms: React Hook Form and Zod
- Context: Zustand
- Routes: React Router Dom
- UI Lib: Radix UI
- Icons: Lucide React
- Charts: Recharts
- Date: DayJS
